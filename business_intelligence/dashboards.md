# Dashboards - NovaBank System

## 1. Overview
Dashboards visualize key metrics and trends for management, operations, and finance teams.

---

## 2. Recommended Dashboards

### 2.1 Executive Dashboard
- Total users and accounts
- Total deposits and withdrawals
- Total transfers (domestic and cross-currency)
- Net balance by currency

### 2.2 Operations Dashboard
- Daily/weekly transaction counts
- Failed or declined transactions
- Accounts with unusual activity
- Pending transfers and audit alerts

### 2.3 Finance Dashboard
- Total balance per currency
- Exchange rate fluctuations
- High-value transfers
- Average account balance trends

### 2.4 User Analytics Dashboard
- New users over time
- Active users by currency
- User retention metrics
- Average transactions per user

---

## 3. Dashboard Features
- Filter by date range
- Drill-down into individual accounts or transactions
- Multi-currency support
- Export charts and data
